# FraudShield X - Render Deployment Guide

This document explains how the FraudShield X Streamlit web app can be deployed using Render.

## Deployment Objective

The goal of this deployment layer is to make the FraudShield X fraud risk web app accessible online through a cloud-hosted environment.

## Deployment Platform

Recommended platform:

- Render

Render is suitable for portfolio projects because it supports GitHub-based deployment and Python web applications.
